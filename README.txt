منصة تحدي اللغة العربية - النسخة الأسطورية

طريقة الرفع:
1. ارفعي index.html ومجلد assets كاملًا إلى مستودع GitHub.
2. تأكدي أن index.html في الصفحة الرئيسية للمستودع، وليس داخل مجلد.
3. من Settings ثم Pages اختاري main و root.
4. انتظري دقيقة، وسيظهر الرابط.

ملاحظة:
هذه النسخة تعمل كذلك مباشرة عند فتح index.html بالمتصفح.
